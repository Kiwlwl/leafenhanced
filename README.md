# LeafEnhanced

Enhances vanilla falling leaves and leaf litter mechanics.

## Features

- **Falling Leaves**: Leaf blocks in deciduous biomes produce falling leaf particles that drift with the wind
- **Leaf Litter Accumulation**: Leaves gradually build up layered leaf litter on the ground
- **Leaf Litter Decay**: Accumulated litter slowly disappears over time
- **Shovel Collection**: Punch leaf litter with a shovel to collect it
- **Full Config**: All features can be toggled and tuned via ModMenu

## Requirements

- Minecraft 26.2
- Fabric Loader >=0.19.3
- Fabric API >=0.154.2+26.2
- Cloth Config >=26.2.155
- ModMenu >=20.0.0 (optional, for config GUI)

## Build

```bash
./gradlew build
```

The built jar will be in `build/libs/`.
